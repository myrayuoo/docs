# Operation

## Opening the screen/backend shell
1. SSH into the server
2. Use the commands below
```sh
cd /var/www/ketamine
sudo screen -ls
```
3. Copy the numbers before the dot from the first line
```sh
sudo screen -r [the numbers you just copied]
```
